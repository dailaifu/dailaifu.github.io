hexo+github博客搭建

Created2022-08-03|Updated2022-08-04
|Post View:

电脑环境搭建

电脑需要安装git和node。

进入git和node的官网进行对应操作系统的软件下载，安装时可以直接点击下一步直到安装位置为止。安装好git之后桌面右击会出现 Git Bash Here

点击 Git Bash Here打开git输入git --version就可以查看git版本号,输入node -v可以查看node的版本号

hexo初始化

创建博客的安装目录，之后右击点击Git Bash Here 进行hexo的安装与初始化
在git bash here中输入npm install -g hexo-cli进行hexo的安装

输入hexo -v进行hexo版本的查看

再输入hexo init进行hexo的初始化

根据网络环境初始化的速度会不同

初始化之后文件夹内容如下

可以在初始化的目录中右击git bash here再输入hexo s进行博客的初始化查看出现 http://localhost:4000 则代表可以进行预览查看

在浏览器中输入 http://localhost:4000 进入到博客页面
![images][https://raw.githubusercontent.com/dailaifu/images/main/images/hexo%20index.jpg]
这样就可以确认搭建完成了

注册github

打开github官网如图，点击Sign up进行注册，注册时注意用户名的建立，后期你的域名为 用户名.github.io

输入密码邮箱后等待接收验证码。输入验证码后注册成功

进入个人主页后首先点击右上角头像在点击your repositories，进入我的项目

进入我的项目后点击new进行新建项目

![images][repositories]
输入项目名称：用户名.github.io 后期也使用这个来访问博客。我这边之前已经新建过了所以会有错误提示。建议勾选下方的Add a README file这个是创建项目描述文件的意思

点击code，默认选择https不需修改点击后面的复制图标复制项目地址

再打开hexo的文件夹中的_config.yml文件可以使用记事本打开。把最后面的内容按照我的进行修改，划横线的位置改为刚刚复制的项目地址进行保存。一定注意后三行的缩进，缩进有问题是会报错的。

在博客的路径右击打开Git Base Here。输入npm install hexo-deployer-git --save进行 hexo-deployer-git 的安装

再输入hexo d进行博客的部署

按照要求输用户名密码，再进入项目地址就会看到新上传的文件，在浏览器输入 用户名.github.io.git就能看到博客已经可以访问了。

部署后优化

域名映射

将域名解析的类型调整为CNAME，解析内容填写 用户名.github.io.git 再在github的项目中创建CNAME内容填写解析的域名。

主题修改

部分hexo主题的github会写需要依赖和安装命令。安装之后在_config.yml中调整theme: 主题名字。要确认themes文件夹内是否有主题文件。如果调整后有报错大概率是主题依赖的问题

博客文章上传

博客文章新建

输入 hexo new [layout] <title> 其中 layout 为文章布局可以不写， title 为文章标题

然后就可以在博客文件夹中看到 source/_posts/文章标题.md 的文件。格式是兼容markdown的。

博客文章生成及上传

输入 hexo g 便可以生成文章，输入 hexo d 便可以上传文章可以连起来写为

这样就可以上传文章了
